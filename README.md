# TOS Helper

> Nessuno legge mai i *Termini di Servizio (ToS)* o le *Privacy Policy* prima di cliccare su "Accetto". Spesso, senza saperlo, acconsentiamo a cedere dati personali, al tracciamento GPS continuo o alla condivisione dei nostri dati con terze parti. 

**TOS Helper** è un'applicazione Android che risolve questo problema utilizzando l'Intelligenza Artificiale per analizzare i testi legali e avvisarti sui reali rischi per la tua privacy.

## Concept
TOS Helper agisce come un esperto in Cybersecurity, GDPR e diritto internazionale. L'app estrae in pochi secondi le clausole pericolose, fornendo un'esperienza personalizzata in base alla tua giurisdizione per garantire il rispetto dei __tuoi diritti__.

## Tecnologie Utilizzate
*   **Ambiente di sviluppo:** Android Studio
*   **Linguaggio:** Kotlin
*   **Integrazione IA:** Qwen API

## Come funziona
Una volta selezionata la propria giurisdizione, l'interfaccia permette con una schermata di input di inserire il TOS da analizzare, e con una schermata di output di visualizzare i pericoli del suddetto TOS:

### 1. Schermata Input
*   **Casella di testo:** Incolla qui la Privacy Policy o i Termini di Servizio (o una parte di essi).
*   **Azione:** Premi il bottone **"Scansiona Rischi Privacy"**.

### 2. Schermata Output
*   **Privacy Score:** Un voto da 1 a 10 sulla sicurezza dei dati.
*   **I Pericoli:** 3 o 4 punti elenco chiari con i rischi principali.
